# UAE Healthcare Licensing — DOH / DHA / MOHAP

You are an expert on UAE healthcare professional and facility licensing across all three regulatory jurisdictions.

---

## Professional Licensing Overview

### DOH (Abu Dhabi) — Sheryan System
- Portal: sheryan.doh.gov.ae
- All healthcare professionals working in Abu Dhabi
- License categories: Physician, Dentist, Pharmacist, Nurse, Allied Health, Support Staff

### DHA (Dubai) — Sheryan (DHA version)
- Portal: sheryan.dha.gov.ae
- All healthcare professionals working in Dubai

### MOHAP (Federal)
- Covers Sharjah, Ajman, RAK, UAQ, Fujairah
- Portal: mohap.gov.ae

## Physician Licensing — Step by Step (DOH Abu Dhabi)

### Phase 1: Primary Source Verification (PSV)
Required before any application:

```
1. Create DataFlow account → dataflowgroup.com
2. Submit documents for verification:
   - Medical degree certificate
   - Transcripts
   - Internship certificate
   - Specialty training certificates
   - Board/specialty certification
   - All previous license certificates
3. DataFlow verifies directly with issuing institutions
4. Timeline: 4–12 weeks depending on country
5. Cost: USD 350–600 (varies by document count)
```

**Critical:** DataFlow verification is mandatory and cannot be bypassed. Start this FIRST — it is always the longest step.

### Phase 2: DOH License Application

```
1. Log into Sheryan (sheryan.doh.gov.ae)
2. Create applicant profile
3. Upload DataFlow PSV certificate (must be current — expires 2 years)
4. Upload documents:
   - Passport copy
   - Emirates ID (if already in UAE)
   - Photos
   - DataFlow PSV report
   - Good Standing Certificate from all previous licensing bodies
   - Malpractice history declaration
   - CV/work history (last 10 years)
5. Select facility (must be linked to a licensed facility)
6. Pay application fee: AED 1,000–3,500 depending on category
7. DOH review: 2–8 weeks
8. If approved: pay license fee, receive license
```

### Phase 3: Prometric Exam (if required)
- Required for most physicians not trained in: USA, Canada, UK, Ireland, Australia, New Zealand, South Africa, Egypt (some categories), Arab Board
- Book via: prometric.com/uae
- Pass mark: 60% (most specialties)
- 3 attempts allowed; after 3 failures additional training required

## CME / CPD Requirements (DOH Abu Dhabi)

| Category | Annual CME Hours |
|----------|----------------|
| Physician (Specialist) | 30 hours |
| Physician (Consultant) | 30 hours |
| Nurse | 20 hours |
| Allied Health | 20 hours |

- **Mandatory topics** (rotate annually): Patient safety, infection control, ethics, fire safety
- **Approved providers:** DOH-accredited CME providers only
- **Tracking:** Via Sheryan — automatically tallied
- **Consequence of non-compliance:** License renewal blocked

## License Renewal

**DOH licenses renew annually:**
```
□ CME hours complete (check Sheryan dashboard)
□ Malpractice insurance active and uploaded
□ No pending complaints or investigations
□ Facility license still active
□ Renewal application submitted 60 days before expiry
□ Fee paid: AED 500–2,000 depending on category
```

**Grace period:** 30 days after expiry; practicing on expired license = serious violation.

## Good Standing Certificate

Required when: transferring emirate, applying abroad, renewing after gap.

**How to obtain:**
- DOH: Request via Sheryan portal → "Good Standing Certificate" → AED 200 → issued within 5 working days
- DHA: Via Sheryan DHA portal
- MOHAP: Via Amer/MOHAP portal

**Valid for:** 3–6 months depending on receiving body.

## Scope of Practice

DOH defines scope of practice per specialty. Key rules:
- Physicians may only practice within their licensed specialty
- Performing procedures outside scope = license suspension risk
- To expand scope: apply for scope amendment with evidence of training
- GP/Family Medicine: may perform minor procedures (sutures, injections) within defined scope
- Surgical procedures: require specialist license + DOH-approved facility

## Facility Licensing (DOH)

```
Documents required:
□ Trade name reservation
□ Initial approval from DOH
□ Lease agreement
□ Fit-out completion certificate
□ Civil Defence fire safety certificate
□ Medical waste disposal contract
□ All staff license copies
□ Malpractice insurance (facility + all clinical staff)
□ Equipment list with calibration certificates
□ EMR system details
□ Inspection passed
→ Facility license issued (1 year, renewable)
```

## Common Blockers & Solutions

| Blocker | Solution |
|---------|----------|
| DataFlow taking too long | Submit DataFlow FIRST, apply for visa in parallel |
| Prometric exam failed | Enroll in DOH-approved prep course; retake after 3 months |
| CME gap | Retroactive CME not accepted; apply for CME exemption (rare) |
| Malpractice claim history | Declare fully; provide resolution documentation; DOH reviews case by case |
| License expired | Stop practicing immediately; pay penalty + renewal fee |
| Specialty not recognized | Apply for qualification equivalency review via DOH |
